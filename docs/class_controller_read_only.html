Detailed Description

A base controller that sets up the logger and the repository, as well as the index view and the details view

Template Parameters
TControllerThe controller that our base class is getting implemented from, this is used for the Logger and the comments
TRepositoryThe type of repository that the controller should implement, should implement IRepository<TModel, TModelID>
TModelThis should be the same model that the TRepository works with

This abstract class is used to make our code cleaner and not write repetitive code as each class implements the same index method and details method logic so there is no need to write more than needed
